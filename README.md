# 🎣 AI 미식 수산물 크레인 오락기 (Fishery Claw Machine)

> **"쫀득한 손맛의 크레인 낚시부터 즉석 가공, 그리고 AI 셰프의 미식 요리 연구소까지 한판에 즐기는 타이쿤 아케이드 게임!"**
> 
> 이 프로젝트는 수산물의 생산, 가공, 미식 요리 과정을 직관적이고 중독성 있는 피지컬 오락기 형태로 구현한 웹 애플리케이션 시뮬레이터입니다.

---

## 🎮 주요 게임 특징

1. **쫀득한 레트로 피지컬 조작**: 
   * 복잡한 모바일용 조작계를 걷어내고, 오락실 감성을 극대화한 **[좌우 화살표 조이스틱]**과 **[GRAB!]** 버튼(혹은 키보드 방향키)만으로 집게를 조종하여 생생하게 움직이는 수산물을 낚아 올립니다.
2. **리얼타임 수중 생태계**: 
   * 꽃게, 문어, 고등어, 조개, 광어 등 총 9종의 풍성한 어종이 등장하며, 바닥 생물과 해초가 흔들리는 깊이 있는 심해 비주얼을 제공합니다.
3. **가공 및 수확 인벤토리**: 
   * 획득한 수산물은 즉석 가공 벨트를 통해 '자반', '자숙', '손질' 상태로 자동 가공되어 우측 인벤토리에 보관됩니다.
4. **🍳 AI 미식 요리 연구소 (Cooking Lab)**:
   * 가공 보관함에서 원하는 재료를 마우스로 콕콕 찝어 가마솥에 담은 뒤 요리 버튼을 누르면, 지글지글 끓어오르는 조리 애니메이션과 함께 고유 요리 카드가 완성됩니다.

---

## 🕹️ 조작 방법 (How to Play)

### 1. 낚시 및 수확 단계
* **조이스틱 버튼 클릭** 또는 **키보드 방향키 `←`, `→`**: 집게 크레인을 좌우로 움직여 조준합니다.
* **`GRAB!` 버튼 클릭** 또는 **키보드 방향키 `↓` (또는 Spacebar)**: 집게를 아래로 내려 물고기를 낚아 올립니다.
* **가공 벨트**: 집게가 물고기를 놓아주면 가공 벨트를 통과하여 우측 인벤토리 상자에 보관됩니다.

### 2. 미식 요리 및 비즈니스 단계
* **냄비에 담기**: 우측 인벤토리에 들어 있는 수확 품목을 마우스로 클릭하면 좌측 가마솥(Cooking Pot)에 재료가 쏙 담깁니다. (최대 2개 조합 가능)
* **레시피 의뢰**: 가마솥 아래의 **`🍳 레시피 의뢰 및 요리하기`** 버튼을 누르면 지글지글 요리가 시작됩니다.
* **납품 및 계약**: 생성된 레시피를 바탕으로 계약 조건에 부합하면 납품을 완료하여 대량의 포인트와 골드를 획득합니다!

## 🔑 Gemini API 연동 시 활성화되는 고급 피처

화면 하단에 **Google Gemini API Key**를 입력하고 **[적용]**을 누르면, 고정된 로컬 데이터를 넘어 무한히 확장되는 차세대 AI 피처가 열립니다. *(API 키가 없어도 기본 내장 로컬 DB로 핵심 게임 플레이는 무한히 가능합니다)*

* **🍳 무한 맞춤형 레시피 창작**: 냄비에 담은 수산물과 유저가 직접 입력한 냉장고 속 부재료(예: 치즈, 라면사리 등)를 실시간으로 결합해 세상에 단 하나뿐인 기상천외한 퓨전 레시피를 AI가 즉석 창조합니다.
* **💼 다이내믹 납품 계약 의뢰**: 전 세계 바이어들이 제안하는 이색적이고 유머러스한 납품 퀘스트가 실시간 JSON 데이터로 계속해서 업데이트됩니다.
